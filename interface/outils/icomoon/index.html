Icomoon

Icomoon est une bibliothèque d'icones.

  1. Télécharger Icomoon
    Télécharger Icomoon
  2. Lier la bibliothèque Icomoon

    Ensuite, lier les feuilles de style CSS avec le code ci-dessous :

    <!-- Liaison CSS Icomoon -->
    <link href="css/icomoon.css" rel="stylesheet">
  3. Utiliser les icones

    Enfin, à l'aide de la documentation vous pouvez utiliser les icones.
